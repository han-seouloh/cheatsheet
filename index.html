HTML

This section is meant to refresh the reader regarding common HTML tags which are used to "mark-up" what type of element the enclosed content is meant to be.

(Remember that HTML stands for Hiper-Text Markup Language)

Common HTML Tags

Tag

Name

Description
<!DOCTYPE html> Doc Type Declares to the browser what type of document to expect.
<head> Head Contains all the metadata such as the title displayed on the browser's tab.
<body> Body Defines the space where text, images, buttons will be displayed.
<div> Division Container that divides the page into sections.
<h#> Heading Defines headings ordered from largest to smallest (# goes from 1 - 6).
<p> Paragraph Defines that the text inside the tags is a paragraph.
<ul> Unordered list Creates a list of items in no particular order using bullets.
<ol> Ordered list Creates a list of items in a particular order (numbered).
<li> List item Adds list items.
<img src=""/> Image Adds image to a webpage (self-closing tag).
<video> Video Adds video to a webpage.
<a> Anchor Used to add links (wrap around any element to turn it into a link).
<!--comment--> Comment Used to add comments to an html file (comments are code lines that are not read by the browser).

Semantic HTML Tags

Tag

Name

Description
<header> Header Container for navigational links or introductory content.
<nav> Navigation Contains block of navigation links such as table of contents.
<link> Link Links CSS file or other resources to HTML file (self-closing).
<main> Main Encapsulates the dominant content within a webpage.
<section> Section Defines sections of content with the same theme (chapters, headings, etc.).
<article> Article Holds content that makes sense on its own.
<aside> Aside Marks additional info that can enhance another element but isn't required to understand the main content. (Bibliographies, endnotes, etc.)
<figure> Figure Encapsulates media such as an image, video, etc. referenced in the main flow of the document.
<footer> Footer Used mainly to display contact info., terms of use, copyright info, etc. at the end of the webpage body.

CSS

This section covers how selectors are declared in CSS and a quick reminder regarding common CSS properties.

CSS Selectors

HTML Tag

CSS Selector

Additional Notes
<all> * {} Universal selector
<tag> tag {} Tag selector
<tag class='class'> .class {} One can add many classes into the html class attribute. ie. <tag class='class class2 class3'> and applies all rulesets pertaining to each defined class.
<tag id='id'> #id {} When an html element needs to be styled uniquely. (id's override all rulesets since it is meant to style uniquely only one specific element)
<tag attribute=''> [attribute] {}, tag[attribute*='value'] {} Targets all elements with the attribute containing the value enclosed with ''.
<tag> at a certain event tag:event/action {} Pseudo-class selectors. Changes targeted element when an event happens.
N/a tag.class {} Chaining. This is used when two or more selectors are needed to target more specific elements.
N/a .class tag {} or tag tag2{} Descendant combinator. Selects elements nested within other html elements. ie. selecting all nested tag elements within the parent html containing a certain class or selecting nested tag2 elements within a parent tag.
N/a tag, .class {} Multiple selectors. Applies the same style to all tag and .class selected.

Common CSS Properties

Property

Description
!important Overrides any style. (This goes after a property declaration.) ie. property: value !important;
color Sets text color (forefround) where the value can be:
  • #000000
  • rgb/a(r, g, b | alpha)
  • hsl/a(degree, saturation, lightness | alpha)
background-color Sets background color.
opacity How transparent an element is from 0 (transparent) - 1 (opaque).
background-image: url('') Set element's background to image.
border Adds border to element.
font-family Sets font. Font must be already installed in the computer.
font-size Sets font size
font-weight Sets font thickness from 1 (lightest) - 1000 (boldest). Not all fonts have numeric weight values available.
font-style Sets font to italic or normal
text-align Aligns text to the element that holds it in any of the following manners:
  • left
  • center
  • right
  • justify
text-transform Changes text to uppercase, lowercase, etc.
